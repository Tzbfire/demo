# Coding Agent / Codex Briefing

A 30-minute HTML slide deck for non-technical office workers: what coding agents are, and what OpenAI Codex is.

## Live Deck

This repository is intended to be deployed with GitHub Pages. Open `index.html` or visit the Pages URL once deployment completes.

## Files

- `index.html`: HTML deck with overview and presentation mode.
- `slides/`: individual 1920x1080 slide pages.
- `shared/tokens.css`: shared visual system.
- `output/coding-agent-codex-briefing.pdf`: PDF handout.
- `output/coding-agent-codex-briefing-visual.pptx`: visual-fidelity PowerPoint, each slide as an image.
- `output/coding-agent-codex-briefing.pptx`: editable approximation.
- `speaker-notes.md`: presenter notes.
- `product-facts.md`: public source notes and safe wording.

## Local Preview

Open `index.html` in a browser. Click `开始演示`, use arrow keys or space to navigate, and press Esc to return to overview.
